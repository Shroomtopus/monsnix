Kurzfassung vorweg: Diese Website kommt ohne Werbung, ohne Tracking und ohne Analyse-Dienste aus. Es werden so wenige Daten verarbeitet wie technisch möglich.

Verantwortlich für die Datenverarbeitung auf dieser Website ist Dave F., Schweiz — erreichbar über das Kontaktformular. Massgeblich ist das Schweizer Datenschutzgesetz (revDSG); soweit anwendbar, wird auch die europäische DSGVO berücksichtigt.

Diese Website wird bei GitHub Pages gehostet, einem Dienst der GitHub Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA. Beim Aufruf der Seiten verarbeitet GitHub technisch notwendige Verbindungsdaten (insbesondere IP-Adresse, Datum und Uhrzeit des Zugriffs, abgerufene Datei, Browser-Kennung), um die Website auszuliefern und die Sicherheit des Dienstes zu gewährleisten. Diese Daten können in den USA verarbeitet werden. Details: Datenschutzerklärung von GitHub.

3. Kontaktformular (FormSubmit)

Nachrichten über das Kontaktformular werden durch den Dienst FormSubmit (formsubmit.co) an mein persönliches Postfach weitergeleitet. Verarbeitet werden die von dir eingegebenen Angaben (Name, E-Mail-Adresse, Nachricht). Die Angaben verwende ich ausschliesslich zur Beantwortung deiner Anfrage; sie werden nicht weitergegeben und nicht für Werbung genutzt. Details: Datenschutzhinweise von FormSubmit.

4. Cookies, Tracking, Analyse

Diese Website setzt keine Cookies, verwendet kein Tracking und keine Analyse-Dienste. Einzig deine Wahl des Anzeigemodus (Tag/Nacht) wird lokal in deinem Browser gespeichert (localStorage) — diese Information verlässt dein Gerät nie und kann jederzeit über die Browser-Einstellungen gelöscht werden.

5. Schriften und externe Inhalte

Alle Schriften werden lokal von dieser Website geladen; es besteht keine Verbindung zu Google Fonts oder anderen Schriften-Diensten. Die Website bindet keine externen Inhalte (Karten, Videos, Social-Media-Widgets) ein, die beim Seitenaufruf Daten an Dritte übertragen.

Du hast das Recht auf Auskunft über die dich betreffenden Personendaten sowie auf deren Berichtigung oder Löschung. Da diese Website selbst keine Personendaten speichert, betrifft dies in der Praxis vor allem Nachrichten, die du mir über das Kontaktformular geschickt hast — melde dich dafür einfach über denselben Weg.
